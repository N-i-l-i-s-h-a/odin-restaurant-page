# Odin Restaurant Page

## Overview
**Odin Restaurant Page** is a dynamic, single-page restaurant website created as an assignment for the JavaScript curriculum in **The Odin Project**. It was a great exercise to practice how to use **Webpack** to bundle modules, assets, and manage development workflow.

## Features
This project includes:
- A dynamic content structure built using **JavaScript DOM manipulation**
- Navigation tabs for Home, Menu, and About pages
- Modular JavaScript files managed with **ES6 modules**
